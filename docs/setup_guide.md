# Setup Guide

## Requirements
- Running n8n instance (self-hosted or cloud)
- Google Sheets / Forms API credentials
- Trello API credentials
- Telegram Bot credentials
- Google Calendar API credentials
- Agile CRM API credentials

## Steps
1. Import workflow JSON into n8n
2. Configure all API credentials
3. Set board/list IDs, chat ID, calendar ID, CRM tags
4. Activate workflow
5. Test with sample submission

## Recommended Settings
- Execution mode: queue
- Error workflow enabled
- Logging enabled for first 48 hours
